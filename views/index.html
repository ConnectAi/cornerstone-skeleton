Welcome

Cornerstone is a production ready NodeJS framework built with ES6 on top of ExpressJS. We built this because we hated everything else. I hope you Enjoy 1.0

To the Docs!

ExpressJS

Express has some killer routing for Node. We take that a step further and make those routes your controllers.

Stylus

To each his own. We use stylus for now, which gets automatically compiled when the CSS file is requested.

Services

CS comes with a way to build an share arbitrary code. This was something that we couldn't find anywhere else.

MySQL

For now, we build in MySQL. But everything is wrapped in Promises, so all of your database code is non blocking.

ES6

Have you used ES6? It will change your JavaScript life. Writing code for CS is a dream.

Debugging

We made Debugging a #1 priority. You can find out what's going on at every step of the way. Check the docs for more.
